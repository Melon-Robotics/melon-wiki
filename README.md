# Melon Wiki

[![pages-build-deployment](https://github.com/Melon-Robotics/melon-wiki/actions/workflows/pages/pages-build-deployment/badge.svg)](https://github.com/Melon-Robotics/melon-wiki/actions/workflows/pages/pages-build-deployment)

An open-source, authoritative knowledge base for commercial diving, subsea robotics, and autonomous maritime operations.

## About

This wiki is built using [Hugo](https://gohugo.io/) with the PaperMod theme and is automatically deployed to [wiki.melon.org](https://wiki.melon.org/).

## Topics Covered

- Commercial Diving
- Subsea Robotics
- Ocean Data & Trust
- Safety & Compliance
- Swarm Systems
- Simulation & Training
- Open Standards
- Glossary

## Building Locally

To build and preview the wiki locally:

```bash
hugo server
```

Visit `http://localhost:1313` in your browser.
